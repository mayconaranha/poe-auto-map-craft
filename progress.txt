## Codebase Patterns
- Use `_resource_path()` helper for any file access so paths work both in dev and PyInstaller .exe
- Images live in `image/` subfolder: alchemy.png, scouring.png, t16.png, t16-5.png
- Template matching uses OpenCV (TM_CCOEFF_NORMED); suppress matched regions by zeroing out the result matrix to find multiple occurrences
- GUI actions run in a daemon thread to keep tkinter responsive; use `self.after(0, ...)` to update UI from the thread
- PyInstaller bundle: `--add-data "image;image"` (Windows semicolon separator)
